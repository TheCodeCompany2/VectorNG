## 💻 About The Project

**VectorNG** is my personal project to master programming. Instead of just reading boring textbooks, I am building a raw motorsport execution loop to calculate racing physics, track car speeds, and eventually map out telemetry graphs using Matplotlib.

### Why the name?
* **Vector:** A nod to both physics (tracking speed and direction into a corner) and graphics (scalable UI assets).
* **NG:** Next Generation (because it sounds awesome).

---

## 🛠️ My Hardware Setup

I am coding and running this project entirely on a lightweight, hardware-constrained environment:

* **Hardware:** Raspberry Pi 400 (Cortex-A72 Quad-core)
* **Storage & OS:** 32GB SanDisk USB Drive (Configured for Direct USB Boot)
* **Language:** Python 3.12 / 3.13
