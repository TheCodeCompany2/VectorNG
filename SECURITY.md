# Security Policy

## Supported Versions

Currently, only the `main` branch is receiving security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in VectorNG, please do not open a public issue. Instead, please email the project maintainer directly. 

We want to express our deepest gratitude to anyone who reports an issue. Thank you for your guidance and for taking the time to help keep this project secure. Your expertise makes VectorNG a better and safer simulator for the whole community!

We will try our best to acknowledge your report within 48 hours and fix the issue as quickly as possible.
