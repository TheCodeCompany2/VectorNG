# Contributing to VectorNG

First off, thank you for considering contributing to VectorNG! It's people like you that make this racing simulator awesome.

## How Can I Contribute?

### 1. Reporting Bugs
If you find a bug (like the car falling through the map, or AI driving into walls), please open an Issue. Describe what happened and how to reproduce it.

### 2. Suggesting Enhancements
Have an idea for a new track, a new car model, or better physics? Open an Issue and tag it as an "enhancement" to start a discussion.

### 3. Submitting Pull Requests
If you want to write code for the project:
1. Fork the repo and create your branch from `main`.
2. Add your new track math or car physics.
3. Test your code to make sure it runs without crashing.
4. Issue that Pull Request!
